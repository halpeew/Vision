# Vision Project
Bilgisayarımızı  klavye ve fareden bağımsız kullanabilmemizi sağlayan bir proje.

# Kullanım
Ellerinizi kullanarak çeşitli komutlar verebilir, çeşitli butonlarla etkileşime geçebilirsiniz.

# Özellikler
-Ses açma kapama için yumruk yapıp, yukarı/aşağı yönlendirdiğinizde ses seviyesi artıp/azalacaktır.
-Bir eliniz ile "Rock-On" yaparken, diğer eliniz ile sola kaydırırsanız, sekmeler arasında geçiş yapabileceksiniz.
-Sol tarafta bulunan HumanVitals butonuna tıklayarak, PyVitals tarafından analiz edilen BPM Değerinize ulaşabilirsiniz.(Beta)
-Sol tarafta bulunan SystemVitals butonuna tıklayarak, sistemin ne kadar yük altında olduğunu görebilirsiniz.


Not: proje geliştirlme aşamasındadır.
